# codealpha-image_gallery
CodeAlpha Internship Task: image_gallery using HTML, CSS, and JavaScript.

## 🔗 Live Demo
https://tanushree-tayde.github.io/codealpha-image_gallery/

## 🛠️ Built With
* **HTML5** – Native `<dialog>` modal layout.
* **CSS3** – Responsive fluid masonry grid.
* **JavaScript** – Vanilla event delegation logic.

## 🚀 Key Features
* **Dark Mode Theme** – Premium dark interface configured using native CSS variables.
* **Smart Filter Navigation** – Next/prev sliders skip hidden images.
* **Native Modal Engine** – Fast, built-in browser rendering.
* **Responsive Breakpoints** – Clean fluid scaling down to mobile screen sizes.

## 💻 How to Run Locally
1. Clone or download the repository.
2. Put your image files in the `images/` folder.
3. Open `index.html` in any modern browser.
